---------------------------------------------------
Duplicate File Deletion
---------------------------------------------------

Description:
This application is used to delete duplicate files from the specified folder/directory after every specific interval as given by user and maintains the log of all the deleted files.
The log file generated is then sent to the through email to the recipient as specified by the user.

Commands:
python program_name.py directory minutes recipient_email_id

eg. python Assignment13_1.py D:\RV 5 adwani708@gmail.com
